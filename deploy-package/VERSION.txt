技术栈:
- 后端: Spring Boot 2.7.18, Spring Data JPA, MySQL
- 前端: Thymeleaf, 内联CSS样式, Emoji图标
- AI服务: 阿里云百炼大模型 (Qwen)
- 爬虫: Jsoup, HttpClient, 智能反爬规避
- 部署: Maven, Java JAR

注意事项:
1. 首次启动会自动创建示例数据
2. 管理员默认账号: admin/admin123
3. 需要配置阿里云百炼大模型API Key
4. 建议使用独立的MySQL数据库
5. 生产环境请修改默认密码
6. 系统已精简为3个核心采集器，提高稳定性
7. 日报生成功能完全稳定，支持并发安全

部署步骤:
1. 确保Java 8+和MySQL已安装
2. 创建数据库: CREATE DATABASE techdaily;
3. 修改application-prod.yml中的数据库配置
4. 运行: java -jar techdaily.jar --spring.profiles.active=prod
5. 访问: http://localhost:8080
6. 进入管理后台测试采集器功能
